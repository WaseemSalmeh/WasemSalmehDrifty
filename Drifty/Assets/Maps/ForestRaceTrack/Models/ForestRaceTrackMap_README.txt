ForestRaceTrackMap
This standalone source model lives in Assets/Maps/ForestRaceTrack/Models and is not applied to the active scene.
The playable prefab lives in Assets/Maps/ForestRaceTrack/Prefabs/ForestRaceTrackMap.prefab.
The Editor builder in Assets/Maps/Editor can rebuild the prefab with Drifty > Maps > Rebuild Forest Race Track Map.
The asphalt object is named Asphalt_Winding_Circuit in the OBJ and Asphalt Winding Circuit in the prefab so the car surface logic can recognize it as asphalt after placement.
